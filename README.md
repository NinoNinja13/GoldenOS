# GoldenOS
GoldenOS — A Liberdade em 1MB
"Adeus empresas controlando eu... Finalmente algo que é realmente meu."

O GoldenOS é um ecossistema modular, leve e independente, desenvolvido para transformar qualquer navegador em um Desktop completo. Criado com a filosofia de soberania digital, ele une a robustez do PHP "antigo" com a inteligência de interfaces modernas.

🚀 O que torna o GoldenOS Único?
📦 Ultraleve & Portátil: Um sistema operacional inteiro em um arquivo único de ~1MB e mais de 7.135 linhas de código puro.

🧠 Inteligência Contextual: Lixeira inteligente, sistema de janelas com snap automático e redimensionamento dinâmico que respeita a Dock.

🎮 Hardware Agnostic: Compatibilidade nativa com Gamepad, Touch e Mouse. Use no PC, no celular ou no console.

⚡ Independência Total: Feito para rodar 24h em micro-servidores de baixo consumo (como Android via KSWEB) sem depender de nuvens de terceiros.

🎨 Personalização Infinita: Suporte a Wallpapers animados via loops JSON e temas dinâmicos.

🛠️ Fábrica de Apps: Crie seus próprios aplicativos dentro do sistema usando HTML/JS e exporte tudo com ícones em Base64 incluídos.

📂 Arquitetura Modular (Open Source)
O GoldenOS é Código Aberto. O núcleo é sólido e inquebrável, mas o sistema é totalmente expansível:

Para Desenvolvedores: Explore o código-fonte em PHP e JS para entender como 1MB pode ser tão potente.

Para a Comunidade: Adicione novos recursos apenas editando arquivos JSON na pasta apps. Transforme o sistema sem precisar mexer no "Kernel".
